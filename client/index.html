Rule Out

Da una prioridad alta a los pacientes que cumplan con los criterios:

intensiveCareInLastMonth true si el paciente estuvo internado en Unidad de Terapia Intensiva en los últimos 2 meses


cancer true si el paciente tiene diagnóstico de cancer


immuneDefficiency true si el paciente tiene algún tipo de inmunodeficiencia


dyspnea true si el paciente refiere tener falta de aire


acuteMotorImpairment true si el paciente tiene dificultad para movilizar algúno de los 4 miembros o para hablar


chestPain true si el paciente tiene dolor de pecho


hematemesis true si el paciente refiere vómitos con sangre


Si el paciente no cumple estos criterios, pasa a la próxima etapa de scoring teniendo en cuenta los síntomas.
